Navigation Pattern (Wizard)

<div class="form-actions">
    <div class="btn-group">
        <!-- Terug button (outline) -->
        <a href="/previous" class="btn btn--outline">
            <svg class="btn__icon">...</svg>
            Vorige
        </a>
        
        <!-- Volgende button (primary) -->
        <button type="submit" class="btn btn--primary">
            Volgende
            <svg class="btn__icon">...</svg>
        </button>
    </div>
</div>

Thymeleaf Fragment (navigation.html)

Voor wizard pagina's met form:

<!-- In form -->
<form method="post">
    <!-- form fields -->
    
    <!-- Navigation fragment -->
    <div th:replace="~{fragments/navigation :: 
         navigation(@{/previous-url}, 'Vorige', 'Volgende', true)}"></div>
</form>

Voor info pagina's zonder form:

<!-- Simple back button -->
<div th:replace="~{fragments/navigation :: 
     backButton(@{/back-url}, 'Terug naar vorige')}"></div>
